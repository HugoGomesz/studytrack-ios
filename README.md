# StudyTrack

Aplicativo iOS para ajudar estudantes a organizar o tempo de estudo, acompanhar progresso e manter hábitos saudáveis com motivações visuais.

## Visão Geral

O **StudyTrack** é um app que oferece ferramentas para agendar sessões de estudo, registrar metas, receber notificações e visualizar histórico.  
Seu diferencial está na gamificação: sistema de streaks (dias consecutivos), XP, níveis e conquistas que tornam o estudo uma jornada envolvente e motivadora.

## Funcionalidades

- Cadastro de sessões de estudo com duração personalizada
- Temporizador com pausa/retomada/cancelamento  
- Histórico de sessões e progresso semanal
- Configuração de metas de estudo
- Notificações locais personalizadas
- **Gamificação:**
  - Tela de perfil com XP, nível, conquistas e recordes
  - Badges automáticos ao cumprir metas e streaks
  - Visualização de streaks (dias seguidos de estudo) e proteção de sequência

## Tecnologias Utilizadas

- Swift  
- Xcode  
- SwiftUI/Storyboard  
- UserDefaults/Core Data  
- Notificações locais  
- Git para versionamento

## Por que iOS?

- Experiência otimizada e fluida, com recursos nativos do sistema
- Notificações confiáveis e análise offline de progresso
- Dados privados e seguros, armazenados localmente
- Foco em expansão futura: widgets, integração com outros apps, gamificação ampliada
